This programm will run the PCC optimizer problem for TSO-DSO coordination.

The main file that needs to be run is named RUN_PCC_optim.m

A prerequisite of being able to run this program, asides of course Matlab is CVX (available at cvxr.com), gurobi and mosek.
All these three packages have academic licenses that are free to download.

All this has been tested with matlab R2020a
